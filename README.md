# ドコゾノ伝説～DX～

## 概要
ジャンル：アクションアドベンチャー<br>
プラットフォーム:Windows<br>
コンセプト:アクションアドベンチャーらしいゲームを作る<br>
<br>
世界に散らばる8つの宝石をあつめるアドベンチャーゲーム。<br>

## アピールポイント<br>
主にDirectX11を使った個人制作の作品シリーズです。<br>
元々のゲーム(ドコゾノ伝説)は市販ゲームハード向けに作成したものでダンジョンは一つでした。<br>
最新版は、ゲームをタイトルからクリアまで一通り作り切ることを目標に、<br>
ステージエディタ―なども作成してダンジョン九つ分のARPGを作りました。<br>

## 制作における問題点・改善点<br>
### 製作時間不足<br>
ARPGとして遊べるほど広さを持った<br>
ステージ制作の時間がとれない。<br>

### 遊びが少ない<br>
市販ハード用の制作の勉強や、DirectX11の勉強と同時並行であったため凝った処理は組めなかった。<br>
結果マップがかなり小さく、敵も二種類しかいなかった。<br>

## 制作での問題点の改善<br>
### 製作時間不足<br>
ステージエディタ―を制作することでステージ制作の時間を確保。<br>

### 遊びが少ない<br>
ステージエディタ―内で個別に当たり判定設定を設けることで見えない壁を作ることを可能にしたり、<br>
最終的にオブジェクトの区別を敵、アイテム、その他に分類しておくことで種類を増やしやすくした<br>

## 操作方法
|  マウス&キー  |  操作  |
| --- | --- |
|  アローキー  |  移動  |
|  A  |  弱攻撃  |
|  S  |  強攻撃・決定  |
|  X  |  回避  |
|  SHIFT  |  移動をダッシュに  |

## エディタ―の操作方法<br>
|  マウス&キー  |  操作  |
| --- | --- |
|  アローキー  |  エリア移動  |
|  WASD  |  カーソル移動  |
|  SHIFT  |  押している間カーソルの移動距離上昇  |
|  E  |  エネミー設置モードに変更  |
|  I(アイ)  |  アイテム設置モードに変更  |
|  1～0  |  エネミーかアイテムの設置・選択  |
|  M  |  選択モードと設置モードの切り替え  |
|  H  |  壁の当たり判定追加  |
|  V  |  水の当たり判定追加  |
|  O  |  マグマの当たり判定追加  |
|  P  |  毒沼の当たり判定追加  |
|  L  |  落とし穴の当たり判定追加  |
|  Q  |  階段の当たり判定追加(押すごとに上り下り・階数が変動)  |
|  G  |  マップチップ設置・選択  |
|  T  |  復活地点移動  |
|  J  |  当たり判定削除  |
|  CTRL+S  |  保存して終了  |
